# CSE1021-Introduction-to-Problem-Solving-and-Programming-Project
Student Management System (SMS)

Overview of the Project-

The Student Management System is a simple Python-based console application designed to manage student records.
It allows users to add, view, update, and delete student details, along with calculating total and average marks.
This project demonstrates the use of Object-Oriented Programming (OOP) concepts in Python.

Features-

1 Add new student details
2 View student information using roll number
3 Update existing student records
4 Delete a student record
5 Calculate total marks of a student
6 Calculate average marks of a class
7 Generate a complete report of all student records
8 User-friendly menu-driven command-line interface

Technologies / Tools Used-

Python 3.12.10
OOP Concepts (Classes & Objects)
Command-Line Interface (CLI)
No external libraries are required.
