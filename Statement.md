Problem Statement-

Educational institutions often face challenges in maintaining accurate and organized student records manually. Tasks such as storing student details, updating information, calculating marks, and generating reports become time-consuming and prone to errors.
This project aims to provide a simple, efficient, and interactive Student Management System that allows users to manage student information seamlessly through a command-line interface.

Scope of the Project-

The scope of this project includes:
Creating a Python-based system to store and manage student details.
Supporting CRUD operations (Create, Read, Update, Delete).
Handling academic data such as marks and subjects.
Calculating total and average marks.
Displaying detailed student reports.
Offering a menu-driven interface for ease of use.
This is a small-scale system ideal for schools, coaching institutes, or personal learning purposes.

Target Users-

This project is designed for:
Teachers who need an easy way to manage student data
School administrators handling academic records

High-Level Features-

Add Student – Insert new student details with name, roll number, class, section, and marks.
View Student – Retrieve complete student information using roll number.
Update Student – Modify existing student details and marks.
Delete Student – Remove a student record from the system.
Calculate Total Marks – Compute the sum of marks for a specific student.
Calculate Average Marks – Find the average marks for an entire class.
Generate Report – Display all student records in a structured format.
Menu-Driven Interface – Simple and interactive command-line navigation.
